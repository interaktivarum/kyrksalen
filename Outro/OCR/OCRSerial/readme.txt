RUN e.g from the command prompt:
---
py copy_and_rename_image_by_serial.py image-file-name

Argument 1: Image file name, the file needs to be located in the Camera folder.

DEPENDENCIES need to be installed:
------------------------------------
OpenCV: https://pypi.org/project/opencv-python/
Python-tesseract: https://pypi.org/project/pytesseract/
